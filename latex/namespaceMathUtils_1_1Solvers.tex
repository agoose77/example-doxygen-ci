Solves non-\/linear equation using Newton method. This function needs two functions, the function to be solved {\ttfamily fun} and its derivate {\ttfamily dfun} 

\begin{DoxyNote}{Note}
The function f(x) must be continues and differentiable. 
\end{DoxyNote}
\begin{DoxyWarning}{Warning}
Throws Non\+Coverge exception when the root is not found.
\end{DoxyWarning}
